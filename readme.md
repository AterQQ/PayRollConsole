# PayRoll Console
The project is created for self-learning on C# syntax. Reference with a simple PayRoll Console from a book called "Learn C# in One Day and Learn It Well". It generates payslips based on the input of the user.

## Requirements
- .NET 9.0

## Usage
A .txt file contains staff information named "staff.txt" in following format
```
Yvonne, Manager
Peter, Manager
John, Admin
Carol, Admin
```

Run the following command in terminal
```zsh
dotnet run
```
